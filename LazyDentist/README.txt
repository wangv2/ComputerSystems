TO COMPILE: 

g++ <name of .cpp file> -lpthread

Example: g++ lazyDentist.cpp -lpthread


TO EXECUTE: 

./a.out <number of chairs> <number of customers> > result
Hit Control C to stop the infinite loops in the program.
Then enter "vim result" to open the text file "result" with the output. 

Example: ./a.out 3 3 > result
Then Control C, following by "vim result" to see the output.
