To compile: 
g++ vowels.cpp -lpthread


To execute: 
./a.out <filepath/> 

For example, to run vowels.cpp with the 20 text files from a folder on my desktop: 
./a.out /Users/vilia/Desktop/files/


NOTES
For executing the program, the user must type “./a.out” followed by the name of the file 
path that contains the text files. It is assumed that the file path ends with a “/“. It is assumed that there are 20 text files which are named as follows: “file1.txt”, “file2.txt”, … “file20.txt”.
